Privacy Policy

Effective Date: 2026

Habiter is developed and maintained by Action Against Hunger ACF. This privacy policy explains how the Habiter application handles user information.

Information Collection

Habiter does not collect, store, or share any personal information from users. All habit data created in the application is stored locally on the user's device.

Data Storage

Habiter stores habit tracking information locally on the user's device using secure local storage. This data remains fully under the control of the user and is not transmitted to any servers.

Third-Party Services

Habiter does not use third-party analytics services, advertising networks, or tracking tools.

Internet Usage

Habiter works completely offline and does not require an internet connection for its core features.

Data Deletion

Users can delete all stored data at any time using the “Delete All Data” option available within the application.

Children's Privacy

Habiter does not knowingly collect personal information from children under the age of 13.

Changes to This Policy

Action Against Hunger ACF may update this Privacy Policy from time to time. Any changes will be reflected on this page.
